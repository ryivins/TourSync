# TourSync

TourSync is a lightweight tour management dashboard for organizing contacts, tasks, and team communication.

## Features
- Contact management with notes and editing
- Task tracking with priorities and due dates
- Messaging system with persistent chat history
- Dashboard overview

## How to Use
1. Open the app:
   https://ryivins.github.io/TourSync/
2. Use the navigation bar:
   - Dashboard → overview
   - Contacts → manage people
   - Tasks → assign work
   - Messages → team chat

## Data Storage
All data is stored locally using browser localStorage.

## Reset Demo
Use the "Reset Demo Data" button to clear all stored information.
